Build a mobile-first AI-powered holistic wellness application called "Natura AI".

GOAL:
A cross-platform mobile app (iOS + Android) that provides natural wellness guidance, personalized plans, habit tracking, AI memory, supplement suggestions, grocery lists, recipe generation, and daily routines.

IMPORTANT:

* This app provides educational wellness suggestions only.
* Include a clear medical disclaimer on onboarding and settings.
* No medical diagnosis, treatment, or medical claims.

---

TECH STACK:

* Mobile: React Native with Expo
* Backend: Node.js + Express
* Database: SQLite (local) + optional cloud (Firebase or Supabase)
* AI: OpenAI API (or mock AI responses if no key)
* State: React Context or Zustand
* Styling: Tailwind (NativeWind)
