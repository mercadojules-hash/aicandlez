## T007 — FINANCIAL CLARITY / TRUST LAYER

Current issue:
Users can still become confused between:

* Paper trading
* Simulated profits
* Lifetime realized P&L
* Recent trade history
* Withdrawable funds
* Live trading eligibility

The app must become radically transparent.

Required changes:

1. Add clarification text anywhere cumulative/lifetime P&L is shown.

Example:
“Realized P&L (Lifetime)”
“Based on all historical closed simulation trades”

2. Add clarification text above Trade History sections.

Example:
“Showing recent closed trades”
“Displaying latest 10 trades”

3. Add clarification under AI Portfolio Manager metrics.

Example:

* Lifetime Realized P&L
* Recent Trades
* Historical Performance
* Simulation Account Metrics

4. Replace dangerous wording:

Current:
“Withdraw Real Profits”

Replace with safer wording:

* “Enable Live Trading”
* “Connect Live Brokerage”
* “Upgrade to Live AI Trading”

DO NOT imply withdrawable real profits unless connected to a funded live brokerage account.

5. Add a persistent simulation disclaimer while in paper mode.

Example:
“Paper Trading Mode — Simulated trades using virtual capital.”

This should appear consistently across:

* Home
* Trade
* Crypto
* Equities
* Profile
