Update Natura AI app background system to use hosted images with proper overlays and positioning.

GOAL:

* Match premium Calm/Headspace style
* Dark readable center
* Light from top-right
* Consistent across all screens

DO NOT:

* Duplicate files
* Change layout structure
* Break existing components

---

1. CREATE BACKGROUND SYSTEM (GLOBAL STYLE)

---

Add this reusable function/helper:

const getBackgroundStyle = (bgUrl) => ({
backgroundImage: `     radial-gradient(circle at center, rgba(0,0,0,0.65) 0%, rgba(0,0,0,0.9) 70%),
    linear-gradient(to bottom, rgba(0,0,0,0.2) 0%, rgba(0,0,0,0.85) 100%),
    url(${bgUrl})
  `,
backgroundSize: 'cover',
backgroundPosition: 'center right',
backgroundRepeat: 'no-repeat'
});

---

2. DEFINE YOUR BACKGROUND URLS

---

const BG = {
