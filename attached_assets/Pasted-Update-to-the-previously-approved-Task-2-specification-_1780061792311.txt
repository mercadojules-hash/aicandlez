Important business rule:

A user can never be both paper and live simultaneously.

Runtime mode is determined by subscription status:

* Free = Paper Only
* Paid Subscription = Live Only
* Subscription Expired/Canceled = Automatically return to Paper Only

Do NOT implement any profit ceiling.

Do NOT stop trading because a user earned too much money.

1. Daily trade allocations
2. Concurrent position limits
3. Existing confidence gates
4. Existing volume gates
5. Existing risk engine

7. Existing EXIT_ENGINE_V2 lifecycle
