1. AI live execution engine

3. LiveTrades persistence UI

* legacy paper-trading mutation

while AI execution itself already uses the live Coinbase adapter.

* AI auto-execution

* OPEN positions count

   Customer runtime is now LIVE-only armed through ACTIVATE AI TRADING.
   Any remaining:

* “paper sim”
* “paper trade”
* “not persisted”
  labels in customer live mode indicate stale legacy rendering paths.

5. Add structured logs
   Need:

* [MANUAL_TRADE_REQUEST]
* [MANUAL_TRADE_EXECUTED]
* [MANUAL_TRADE_REJECTED]
  with:
* normalized symbol
* exchange
* execution mode

When user manually clicks BUY:

* trade executes live
* position persists
* Live Trades panel updates
