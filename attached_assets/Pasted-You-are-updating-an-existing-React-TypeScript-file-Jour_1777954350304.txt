### 4. Unrealized PnL (OPEN trades)

Use current price from:

```ts

```

Simulate current price safely (no API change):

```ts
const getLivePrice = (t) => t.price; // placeholder for now
```

Then:

```ts
const unrealizedPnL = openTrades.reduce((sum, t) => {
  const current = getLivePrice(t);
  const entry = t.price;

  const pnl =
    t.side === "BUY"
      ? (current - entry) * t.amount
      : (entry - current) * t.amount;

  return sum + pnl;
}, 0);
```
